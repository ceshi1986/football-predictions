## 二、Elo模型（当前实际使用的模式）

### 公式
```
d = 主队实力分 - 客队实力分
hf = 0.5 / (1 + 10^(-d/14))    # 主胜概率
df = 0.28 * exp(-|d|/18)        # 平局概率
af = 1 - hf - df                # 客胜概率
归一化使三项之和=1
```

### 球队实力库
- 覆盖138支球队（含五大联赛、挪超、瑞典超、芬超、日职、韩职、美职、巴甲等）
- 实力分范围：62-94
- 未在库中的球队默认70分

## 三、置信度计算

### 基础公式
```
sp = max_prob - second_prob（概率差/凯利离散度）
ct = round((0.4 + sp*0.6 + handicapBonus*0.08) * 100)
```

### 凯利离散度调节
- sp < 0.05 → +15%
- sp 0.05-0.10 → +5%
- sp > 0.15 → -15%

## 四、星级评定

### 无赔率时（当前模式）
- sp > 0.50 → ★★★★★
- sp > 0.40 → ★★★★
- sp > 0.28 → ★★★
- sp > 0.15 → ★★
- 其他 → ★

### 让球辅助
- 让球方向与预测方向一致 → +1星，handicapBonus=1

## 五、单/双选判断

### 单选条件（全部满足）
- max_prob >= 0.60
- sp >= 0.20
- 最低赔率 >= 1.40

### 双选（其余所有情况）
- 第一选择：概率最高的方向
- 第二选择：
  - 有赔率时：排除平局后赔率最高的方向
  - 无赔率时：默认排除平局（主胜选负，客胜选胜）

## 六、Skip规则（不建议购买）

- 最低赔率 <= 1.25 → 赔率过低，投注价值极低
- sp < 0.08（且未被skip） → 结果太不确定
