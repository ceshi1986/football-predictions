## 一、决策流程

```
有赔率？→ 是 → 赔率隐含概率（凯利公式）+ 凯利七场景修正
         → 否 → 降级为Elo模型（纯实力评分）
```

赔率数据断裂时（如当前），全部降级为纯Elo模式。

## 二、凯利指数判断铁律

### 2.1 核心判断标准
- **Kelly ≤ 实时赔付率（返还率）→ 看好该方向**
- **Kelly > 实时赔付率 → 不看好该方向**
- 术语：只叫"实时返还率"，严禁"真实""近似""平局"等变体

### 2.2 数据源
- **主数据源**：zgzcw.com 百家欧赔（覆盖所有联赛，数据可靠）
- **辅助数据源**：500.com（部分联赛如韩职不覆盖，且凯利数据可靠性存疑）
- **核心公司**：**只看Bet365和韦德**做凯利一致性比较（2026-07-21用户明确）

### 2.3 赔率优先规则
- 看好某队不败时，胜vs平之间比赔率，赔率低的排前面（标"可博"）
- 仅当平赔<胜赔且两家该队胜Kelly各自都最低时，才把胜排在平前面

## 三、凯利七场景体系

### 场景①：两家不看好同一方向
- 两家都看好同一队不败（胜平Kelly都≤返还率）
- 判断：**该队不败，平局递进**
- 双选：胜+平（或负+平），可标"可博"

### 场景②：各不看好不同方向（信号矛盾）
- Bet365看好A队不败，韦德看好B队不败（或反过来）
- 判断：**平局高发**
- 双选：平为主，次选赔率更低的方向
- **不标"可博"**——没有一致的看好信号支撑

### 场景③：去平局
- 两家都只看好胜或负（平局Kelly>返还率）
- 判断：**去平局**，胜/负分明
- 单选或双选（排除平局）

### 场景④：独低于返还率加强
- 只有一家看好某方向，另一家一般看好
- 判断：**加强信号**，可信度较高

### 场景⑤：信号矛盾但平局可丢
- 两家看好方向不同，且平局Kelly都>返还率
- 判断：**平局可丢**，胜/负方向选概率高的
- 注意：如果平Kelly不满足条件，平不能丢

### 场景⑥：两家不看好+平Kelly最低
- 两家都看好某队不败，且平局Kelly在两家都是最低值
- 判断：置信度>70% → **平+胜**；否则 → **平+负**

### 场景⑦：蛙跳盘（最强信号）
- 连续跳级变盘 + 凯利一致性信号
- 判断：**最强信号**，可单选
- 但离散度>0.10则放弃，降为双选

### 场景零：共同看好（赔率定主方向，Kelly定补防）
- 两家核心公司对同方向的看好程度一致
- 赔率概率最高方向为主选，其余commonFavor方向为cover

## 五、Elo模型（赔率断裂时降级使用）

### 公式
```
d = 主队实力分 - 客队实力分
hf = 0.5 / (1 + 10^(-d/14))    # 主胜概率
df = 0.28 * exp(-|d|/18)        # 平局概率
af = 1 - hf - df                # 客胜概率
归一化使三项之和=1
```

### 球队实力库
- 覆盖138支球队，实力分范围62-94
- 未在库中的球队默认70分

## 六、置信度计算

```
sp = max_prob - second_prob
ct = round((0.4 + sp*0.6 + handicapBonus*0.08) * 100)
```

### 凯利离散度调节
- sp < 0.05 → +15%
- sp 0.05-0.10 → +5%
- sp > 0.15 → -15%

---

## 七、星级评定

- sp > 0.50 → ★★★★★
- sp > 0.40 → ★★★★
- sp > 0.28 → ★★★
- sp > 0.15 → ★★
- 其他 → ★
- 让球方向与预测方向一致 → +1星

---

## 八、单/双选判断

### 单选条件（全部满足）
- max_prob >= 0.60
- sp >= 0.20
- 最低赔率 >= 1.40

### 双选
- 第一选择：概率最高的方向
- 第二选择：排除平局后赔率最高的方向（有赔率时）；默认排除平局（无赔率时）

---

## 九、Skip规则

- 最低赔率 ≤ 1.25 → 赔率过低，skip
- sp < 0.08 → 结果太不确定，skip

## 十六、联赛特殊规律

### 韩职联赛（K联赛）特征（2026-07-22 记录）

| 特征 | 分析 |
|------|------|
| **平局率高** | 竞争激烈，强弱差距不大，冷门多 |
| **强队不稳定** | 排名前列的球队也容易翻车 |
| **场景②（信号矛盾）→平局高发** | 已验证：两家看好方向不同时，平局概率极高 |
| **场景①（单选）风险大** | 两家都看好但Kelly值接近返还率时，信号不够强，考虑补防平 |
| **场景③（去平局）→次选命中率高** | 去平局后，胜/负方向中赔率较高的那个更容易中 |

### 韩职策略调整
1. **韩职场次优先双选**：平局率太高，单选容易翻车
2. **场景①谨慎单选**：Kelly值与返还率差距<0.05时，建议补防平局
3. **场景②的平局判断很准**：信号矛盾→平局高发，这个规律在韩职特别明显
4. **场景③的次选价值大**：去平局后，次选命中率高
5. **临近赛前凯利转向→强信号**：赔率看好一方，但临近赛前凯利指数转向另一方（如蔚山vs仁川：赔率主胜2.17<客胜3.22，但赛前Kelly看好客胜），这种场次命中率很高
